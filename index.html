Guides

Installing SBSettings themes

1. Drop your zip in /private/var/mobile/Library/SBSettings/Themes

2. Decompress it, then delete the zip

3. Open SBSettings

4. Apply the theme

5. Respring and you're done.

 

Installing Themes, in general.

1. Drop your zip in /Library/Themes

2. Decompress it, then delete the zip

3. Open Winterboard

4. Apply the theme

5. Respring and you're done.

 

Installing apps on any iOS version

BEFORE ANYTHING, INSTALL APPSYNC!

1. Obtain/Install appsync on your respective iOS version (iPhoneOS 2 idk, iPhoneOS 3: repo: ios3.party, iOS 4: https://cydia.akemi.ai/debs/net.angelxwind.appsync40plus.deb, you will need to pass it through archive.org, for iOS 5 and up use the one located at cydia.skyglow.es)

 

(Windows version)

1. Download AppManager from Here (Windows only)

2. Plug in your device and sideload .ipa files.

(iPhoneOS 3.X - 4.X, in-device)

1. Get the repo yzu.moe/dev/

2. Install install0us (make sure it's not 5)

3. Move your IPAs to the specified location inside the app.

(iOS 5.X to 8.4.1, in-device)

1. Get the repo yzu.moe/dev/

2. Install Veteris

3.(ALT) Install Installous 5 to sideload custom IPAs that veteris might not have.

(iOS 9 - Any, MacOS and Windows)

1. Install Sideloadly!

2. Plug your device to your computer and sideload an app by clicking the file icon.

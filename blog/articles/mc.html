First, i visited Macintosh Garden to search for any kind of minecraft ports, and luckily, i found a few.


ClassiCube


Everyone knows ClassiCube, it's a classic, really. It's a sort of "port" of Minecraft Classic to C, which means that it can practically run on anything (DOS, Mac Classic, OSX, etc.), it's not bad, but you're stuck playing Minecraft Classic for the rest of your life as there is no survival mode or mods (as far as i know).


ACTUAL Minecraft


To ACTUALLY play Minecraft, you can visit here and search for a Minecraft version that suits you best (personally i use 1.2.5), download the zip and run the install script!


Installing custom versions


To install custom versions, you first need to download your desired client.jar, then extract it, and finally replace the contents of the /Resources/Minecraft/bin/minecraft.jar for the contents of your client.jar. you might need newer java versions to run newer Minecraft versions.

Codepad

Codepad is an HTML, CSS, JS, image and (surprisingly) lua IDE for iPhoneOS 3.2 (iPad ofc) up to anything the iPad 1 supports i guess. it also has a buggy image editor that uses the canvas tag.

Fun Fact: the app is basically vscode but for ipads. it has syntax highlighting, proper identation and bracket closing, and the actual text editor is built on html.

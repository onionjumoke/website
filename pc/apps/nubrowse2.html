nuBrowse2

nuBrowse2 is a tabbed web browser for jailbroken iOS 4.2+. Features tabbed browsing with up to 8 tabs, DuckDuckGo or Wiby search, 7 user-agent presets, bookmarks, history manager, and desktop mode toggle. All toolbar icons are programmatically drawn using Core Graphics.

This app is compatible with iOS 4.2 and up.
